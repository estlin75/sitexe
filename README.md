# TestForge - Elite Cyberpunk Landing Page 🚀

Ekskluzywna wizytówka Agencji AI / Web Dev w stylu Awwwards, Matrix, Mr. Robot. Zbudowana w **2026 roku** na Next.js 15 (App Router), React 19, Tailwind v4 i React Three Fiber.

## 🔥 Stack technologiczny
- Framework: Next.js 15 (App Router) + React 19
- Styling: Tailwind CSS v4 (z własnymi CSS Variables w globals.css)
- 3D & Efekty: `@react-three/fiber`, `@react-three/postprocessing`, customowy deszcz matrixa na Canvas API
- Animacje: Framer Motion (`motion/react`)
- Scroll: Lenis
- Formularze: React Hook Form + Zod
- Powiadomienia (Toast): Sonner
- Ikony: Lucide React
